Introduction

A data structure is used to organize data in a computer's memory so that user can use it effectively. Through this data structure user can store data in a efficient way and can perform tasks like, acess data, insert, delete, update etc. Basically It is a set of algorithms that user can use in any programming language to structure the data in the computer's memory.As a example we can say in any programming language we use array, linked lists or stack to store the data and perform the desired tasks. Suppose user want to store some student's marks.So in this case user can use array data structure to store those data.

In this 1.1 diagram elements are the studen's marks and indexes are the memory location where student's marks are alloted.

There are two types of data structures:

1.Primitive data structure

2.Non-primitive data structure

Primitive Data structure: The primitive data structures are primitive data types. The int, char, float, double, and pointer are the primitive data structures.These data structures can hold a single value.The primitive data structured are:Integer ,char, float, pointer etc.

Non Primitive Data structure:Non Primitive Data structures are derived from primitive Data structures.These datatypes are used to store group of values.The non-primitive data structures are:Arrays,Stack,Queue,Tree,Graph etc.
